# Welsh Flashcards

This is an experiment with creating a Flashcard app using LLMs. Ultimately, although the project was
seeded using an LLM, it turned out to be easier to finish it by hand.

See it at [https://rjw57.github.io/welsh-flashcards/](https://rjw57.github.io/welsh-flashcards/).

## Getting started

```sh
yarn install
yarn run dev
```
